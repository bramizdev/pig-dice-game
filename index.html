Pig Game

Rules

  • Players take turns with a die.
  • On a player's turn he/she can roll a die as many times as they like.
  • If a roll is a 2, 3, 4, 5, or 6, the player adds that many points to their score for the turn
  • A player may choose to end their turn at any time and ‚Äúhold‚Äù their points.
  • If a player rolls a 1, they lose all their unholded points and their turn is over.
  • The first player to accumulate 100 or more points wins the game.

About

Pig Game is an small project created to practice DOM manipulation using vanilla Javascript, CSS and HTML. This project is part of Jonas Schmedtmann's The Complete JavaScript Course 2022: From Zero to Expert! course as an additional feature this project is fully responsive.
